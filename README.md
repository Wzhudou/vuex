### 1、使用Vuex统一管理状态的好处
```
(1) 能够在vue想中集中管理共享的数据，易于开发和后期维护
(2) 能够高效地实现组件之间的数据共享，提高开发效率
(3) 存储在vuex中国的数据都是响应式的，能够实时保持数据与页面的同步
```
